To get started for offline (can load without wifi) playing, download the file from Google Drive (link below). Once at the
Drive page, look for a button next to your profile picture (top right) that looks like a down arrow. Click that and it should download.
Once it's done, just double click on the file in the tote or on the file explorer. For online play, we recommend
using EaglercraftX (newest/safest, link below). If you would like to use any other client, you can search through
the wide amount of clients we have on Eaglercraft Extras (link below). Have fun!
